# Future Improvements
This document outlines potential improvements that could be implemented to increase the performance and functionality of the
robotic arm.
## Increased Base Stability
The stability of the base could be increased by increasing the weight of the base or rigidly attaching the base to a stable surface.
## Computer Vision Integration
Adding a camera and computer vision system would allow the system to detect objects and pick and place them, rather then 
rely on known positions.
## Automated Path Planning
Currently shoulder and elbow movement are performed sequentially, by adding path planning the arm would be able to move to certain locations in the most efficient manner with smoother motion.
